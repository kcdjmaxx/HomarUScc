- **Summary:** Browser automation library supporting Chromium, Firefox, and WebKit. HomarUScc uses Playwright's chromium.launch() for ephemeral sessions and chromium.launchPersistentContext() for persistent browser sessions with cookies/localStorage. Provides page.goto(), page.click(), page.fill(), page.evaluate(), and accessibility snapshotting.
